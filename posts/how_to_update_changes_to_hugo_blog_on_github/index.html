How to update changes to Hugo blog on GitHub

Author
 2021-10-07  141 words   One minute 
Contents
  1. Make sure the local source files of the website are updated with the changes done on GitHub (if any). Skip this step if none.

Go to Terminal

	cd my_site
	git pull origin master
  1. Make any changes to the files locally or add new posts. Go to my_site folder
	hugo new posts/second_post.md 

Open second_post.md using text editor and remove the property draft: true and replace it with draft: false
  1. Launch the website locally to check the updates are correct before updating to GitHub
	hugo serve

Go to http://localhost:1313
  1. Create the public folder and generate the necessary files required for the website in it.
	cd my_site
	hugo
  1. In terminal, go to the Public folder inside my_site folder, run the below commands,
	git add .
	git commit -m “second commit”
	git push origin master

  1. Finally, check out your website on Github

    https://[username].github.io

References:
