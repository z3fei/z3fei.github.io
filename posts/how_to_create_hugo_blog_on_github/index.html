How_to_create_hugo_blog_on_github

Author
 2021-10-07  501 words   3 minutes 
Contents

This is a guide for Mac but similar logic should be same for Windows

  1. Install hugo package 

     brew install hugo

  2. Create the blog. Open your terminal and type in 

     hugo new site my_site
 cd my_site

    my_site will be a new folder and also the name of your site.

  3. Pick a blog theme from Hugo. Go to https://themes.gohugo.io/

  4. Install the theme you like. For this example, I will pick LoveIt theme 

     git clone https://github.com/dillonzq/LoveIt.git themes/LoveIt

    Copy the theme git url, eg. https://github.com/dillonzq/LoveIt.git

    themes/LoveIt = means I am creating and installing the theme in LoveIt folder inside themes folder

  5. Next, you need to find config.toml inside my_site folder and make necessary changes. Open the file using text editor and replace the content with below to point to LoveIt theme, together with some basic configurations. If you choose other theme, it will be different. Follow their instruction on what to change / add in config.toml. The file is required to load and display the theme. 

     baseURL = "http://example.org/"
# [en, zh-cn, fr, ...] determines default content language
defaultContentLanguage = "en"
# language code
languageCode = "en"
title = "My New Hugo Site"

# Change the default theme to be use when building the site with Hugo
theme = "LoveIt"

[params]
  # LoveIt theme version
  version = "0.2.X"

[menu]
  [[menu.main]]
    identifier = "posts"
    # you can add extra information before the name (HTML format is supported), such as icons
    pre = ""
    # you can add extra information after the name (HTML format is supported), such as icons
    post = ""
    name = "Posts"
    url = "/posts/"
    # title will be shown when you hover on this menu link
    title = ""
    weight = 1
  [[menu.main]]
    identifier = "tags"
    pre = ""
    post = ""
    name = "Tags"
    url = "/tags/"
    title = ""
    weight = 2
  [[menu.main]]
    identifier = "categories"
    pre = ""
    post = ""
    name = "Categories"
    url = "/categories/"
    title = ""
    weight = 3

# Markup related configuration in Hugo
[markup]
  # Syntax Highlighting (https://gohugo.io/content-management/syntax-highlighting)
  [markup.highlight]
    # false is a necessary configuration (https://github.com/dillonzq/LoveIt/issues/158)
    noClasses = false

  6. Create your first post. The command below will create posts folder under content folder inside my_site folder. eg. my_site>content>posts>first_post.md

	hugo new posts/first_post.md 

Open first_post.md using text editor and remove the property draft: true and replace it with draft: false
  1. Launch the website locally
	hugo serve

Go to http://localhost:1313
  1. Create the public folder and generate the necessary files required for the website in it.
	cd my_site
	hugo

  1. Create your Github Repository

    To publish the website on Github, you need to create a new repository named [username].github.io

    [username] is your GitHub login id.

    Remember not to initialise this repository with a README and select Public.

  2. In terminal, go to the Public folder inside my_site folder, run the below commands,

	git init
	git add .
	git remote add origin https://github.com/[username]/[username].github.io.git
	git commit -m “first commit”
	git push origin master

  1. Finally, check out your website on Github

    https://[username].github.io
